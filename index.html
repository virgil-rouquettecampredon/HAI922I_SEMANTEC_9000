Pour écrire des règles :
  • Il vous faut utiliser un espace entre chaque opérateurs
  • Les règles se décomposent en deux parties, avec un "=>" pour séparer les hypothèses et les conclusions
  • Pour séparer chaque règles vous pouvez utiliser ";"
  • Pour séparer des hypothèses ou des conclusions vous pouvez utiliser "&"
  • Vous pouvez déclarer une variable avec "$x" ou x est un caractère quelconque
  • Vous pouvez spécifier le type d'un terme ou d'un groupe avec "==" ($x == GN ou $x == Adj)
  • Vous pouvez négativer des relations en préfixant "!" à la relation (par exemple : $x !r_succ $y)
  • Vous pouvez négativer des typages (de fonctions grammaticale ou de groupes) avec "!="
  • Vous pouvez créer un groupe de mots avec des "+" (sans espaces) "$x+$y == GN"

Voici un exemple :
